Liste non exhaustive

  • Événements standards :
    • change : Un élément perd le focus et sa valeur a changé depuis l'acquisition du focus.
    • click : Un bouton d'un dispositif de pointage a été appuyé ou relaché sur un élément.
    • keydown : A key is pressed down.
    • keypress : A key is pressed down and that key normally produces a character value (use input instead).
    • keyup : A key is released.
    • mousedown : A pointing device button (usually a mouse) is pressed on an element.
    • mouseenter : A pointing device is moved onto the element that has the listener attached.
    • mouseleave : A pointing device is moved off the element that has the listener attached.
    • mousemove : A pointing device is moved over an element.
    • mouseout : A pointing device is moved off the element that has the listener attached or off one of its children.
    • mouseover : A pointing device is moved onto the element that has the listener attached or onto one of its children.
    • mouseup : A pointing device button is released over an element.
    • scroll : The document view or an element has been scrolled.
    • submit : A form is submitted.
  • Événements importants :
    • DOMContentLoaded : Le document a fini de charger (mais pas les ressources qui en dépendent).
    • abort : Une transaction a été interrompue.
    • error : Le chargement d'une ressource a échoué.
    • offline : The browser has lost access to the network.
  • Timers :
    • setInterval : Appelle une fonction de manière répétée, avec un certain délai fixé entre chaque appel.
    • setTimout : Exécute un morceau de code ou une fonction après un délai determiné.
    • requestAnimationFrame : The Window.requestAnimationFrame() method tells the browser that you wish to perform an animation and requests that the browser call a specified function to update an animation before the next repaint. The method takes as an argument a callback to be invoked before the repaint.

Réagir à un événement

En JavaScript on réagit à un événement toujours avec une callback (une fonction de rappel) qui peut être anonyme ou non.

Sans jQuery

document.querySelector('div').onclick = function()
{
	alert('click !');
};

Avec jQuery

$('div').click(function()
{
	alert('click !');
});
