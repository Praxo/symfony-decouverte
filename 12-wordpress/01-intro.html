Wordpress : CMS

Un CMS (Content Management System) est un logiciel destiné à la conception et à la mise à jour dynamique de sites Web.
Il offre diverses fonctionnalités telles que l'ajout d'articles, de pages, la gestion des galeries multimédias, formulaires etc etc...

Wordpress est le CMS le plus utilisé, il est gratuit, libre, développé en PHP et repose sur une base de données MySQL.

Initialement créé en tant que moteur de blogs, ses fonctionnalités et la communauté qui l'entoure ont largement élargis ses capacités (sites web, e-commerce, logiciel de gestion...).

Installation de Wordpress

  • Télécharger le package : wordpress (FR)
  • Décompresser dans le dossier "site" servant de base au serveur apache (exemple http://localhost/wordpress/)
  • Créer une base de données (nom : wordpress par exemple) : http://localhost/phpmyadmin
  • Allez sur http://localhost/.... (chemin vers wordpress)
  • Suivez le processus d'installation...
