Langages et technologies

Objectifs

Découvrir les langages et les technologies abordés durant la formation

  • HTML (HyperText Markup Language)
  • CSS (Cascading Style Sheets)
  • Javascript
  • jQuery
  • PHP (Hypertext Preprocessor)
  • SQL (Structured Query Language) et SGBD (Mysql)

HTML (HyperText Markup Language)

Langage de balisage pour gérer et organiser le contenu (c'est un titre, c'est un paragraphe, c'est une image...). 

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title>Hello world</title>
	</head>
	<body>
		<h1 class="titre-principal" id="titre-hello">Hello world !</h1>
	</body>
</html>

CSS (Cascading Style Sheets)

Styliser et gérer l'apparence d'une page (agencement, couleurs, formatage du texte...). 

h1.titre-principal{
	font-family: Arial, Verdana, Sans;
	color: #333;
	text-transform: uppercase;
	text-align: center;
	margin-bottom: 20px;
	font-size: 2em;
}

Javascript

Langage de script interprété et exécuté par un navigateur. Permet (entre autre) de traiter les interactions avec l'utilisateur. 

var mainTitle = document.getElementById('titre-hello');
mainTitle.style.color = "#f00";

jQuery

Bibliothèque (ensemble de fonctions) JS permettant (entre autre) de manipuler les éléments HTML et CSS. 

$(function(){
	$('#titre-hello').css("color","#f00");
});

PHP (Hypertext Preprocessor)

Langage serveur permettant (entre autre) de générer de l'HTML, intéragir avec une base de données... 

<?php
	$color = array("rouge"=>"#f00", "bleu"=>"#00aeff");
	echo $color["rouge"];
?>

SQL et SGBD (Mysql)

SQL (Structured Query Language) : langage permettant de gérer les informations stockées dans une BDD (ex : identifiant / mot de passe).
SGBD : Système de Gestion de Base de Données (phpmyadmin). 

select * from login where email="test@test.fr" and password="motDePasse"
