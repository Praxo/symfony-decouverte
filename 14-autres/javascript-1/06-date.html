Les dates - Documentation - Documentation 2

En JavaScript il est possible de manipuler une date et/ou une heure avec une variable de type objet de la classe Date.

Déclarer et instancier une date

var maintenant; // déclaration d'une variable maintenant
maintenant = new Date(); // de type objet de la classe Date

// maintenant.getDay() renvoie le numéro du jour de la semaine, 0 = dimanche, 6 = samedi.
// maintenant.getDate() renvoie le jour dans le mois (1 à 31).
// maintenant.getMonth() renvoie le numéro du mois dans l'année, 0 = janvier, 11 = décembre.
// maintenant.getFullYear() renvoie l'année.
