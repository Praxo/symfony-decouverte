Découverte du javascript

Objectifs

Découverte du javascript

  • Ecrire du HTML dans la page
  • Ecrire dans la console de développement
  • Créer une alerte
NB : je ne précise pas forcément les balises <script> autours du javascript (flemingite aigue).

Ecrire du HTML dans la page

La méthode document.write() écrit à l'emplacement où on le déclare.

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="utf-8">
    <title>JavaScript - Exemple 1</title>
</head>
<body>
    <h1>Titre de la page</h1>
    <script>
        document.write('<p>Bonjour tout le monde !</p>');
    </script>
</body>
</html>

Ecrire dans la console de développement

La méthode console.log() écrit le message dans la console de développement du navigateur web (F12).

console.log('hello console !');

Très utilise pour afficher des informations durant la phase de développement.

Créer une alerte

La méthode window.alert() écrit un message dans une fenêtre modale affichée par le navigateur.

window.alert('hello popup !');
