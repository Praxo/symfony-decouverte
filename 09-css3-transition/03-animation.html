Animations CSS3

Les animations CSS3 sont semblables aux transitions mais permettent un contrôle très précis dans le temps.
Une animation peut être lancée lors d'un changement d'état, un ajout de class en JS ou au chargement de la page.

Une animation se fait en 2 étapes :

  • Créer l'animation (genre de "fonction")
  • Assigner l'animation à un sélecteur

Définir une animation

  • Une déclaration d'images-clés (keyframes)
  • Le nom de la déclaration (comme une fonction)
  • Les différentes étapes de l'animation
/* Règle CSS3 donc on ajoute les prefixes */
@-moz-keyframes monanim {}
@-webkit-keyframes monanim {}
@-o-keyframes monanim {}
@-ms-keyframes monanim {}

/* déclaration de keyframes et nommage */
@keyframes monanim {
	0%{
		/* état initial (obligatoire) */
		color: red;
	}
	50%{
		/* état à 50% */
		color: green;
	}
	100%{
		/* état final (obligatoire) */
		color: red;
	}
}
NB : Une transition est effectuée entre chaque keyframe.

Assigner une animation

Une fois l'animation créée, il faut l'appeler avec animation (CSS3 donc préfix).

Liste des paramètres :

  • animation-name : nom de l'animation (obligatoire)
  • animation-duration : durée en s ou ms (obligatoire)
  • animation-iteration-count : nombre de fois que l'animation est effectuée (défaut 1), infinite si l'animation se répète à l'infini
  • animation-direction : définit si l'animation se joue toujours du début à la fin ou si on alterne (défaut normal), alternate pour de "l'aller-retour" (exemple)
  • animation-timing-function : idem transition, effet de timing
  • animation-delay : délai

On peut Ou alors, utiliser le raccourci animation

.bouton{
	animation: slideIn 3s infinite alternate;
}

@keyframes slideIn{
	0%{
		left: 100%;
	}
	100%{
		left: 0%;
	}
}
