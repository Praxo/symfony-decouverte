Déclarer un formulaire

La balise form englobe les éléments d'un formulaire HTML (obligatoire).
L'intérêt d'un formulaire est d'envoyer des informations (pour recevoir un mail, enregistré en base de données...). Le traitement du formulaire se fait en php.

  • method (get | post) : méthode d'envoi des informations
  • action : lien de la page / du fichier qui va traiter les informations (envoie l'email, enregistre en BDD...)
  • enctype="multipart/form-data" : attribut obligatoire lors de l'upload d'un fichier
<form method="post" action="traitement.php" enctype="multipart/form-data">
	<!-- Contenu du formulaire -->
</form>
NB : Je ne vais pas indiquer la balise form par la suite

Champs de saisie

L'attribut name permet de nommer le champ, et de récupérer sa valeur dans le traitement (principe de variable).

Zone de texte uni-ligne

<input type="text" name="pseudo" /> <!-- pseudo aura comme valeur, ce que l'utilisateur aura renseigné -->

Mot de passe : input password

<input type="password" name="mdp" />

Zone multiligne : textarea

<textarea name="description"></textarea>

Autres attributs possibles

  • maxlength : nombre de caractères max que l'utilisateur peut saisir
  • value : pré-rempli le champ par valeur par défaut
  • placeholder : indication sur le contenu (disparait au focus)
  • required : champ obligatoire (html5)
  • disabled : champ désactivé

Libellé de champ

label est une balise de libellé. Elle va permettre de lier un texte à un champ (lors du click sur le texte, le contenu aura le focus).
Pour lier un libellé à élément de formulaire, il faut ajouter des attributs de valeur identique :

  • label : attribut for
  • input : attribut id
<label for="champPseudo">Pseudo : </label> <input type="text" name="pseudo" id="champPseudo" />

Éléments d'options / choix

Cases à cocher (unique) : input radio

  • value permet d'indiquer la valeur du champ
  • name est identique pour lier les champs (cocher l'un -> décoche l'autre)
  • checked coche une case par défaut
Je suis :
<input type="radio" name="choixSexe" value="homme" id="choixHomme" checked /> <label for="choixHomme">un homme</label>
<input type="radio" name="choixSexe" value="femme" id="choixFemme" /> <label for="choixFemme">Une femme</label>
<input type="radio" name="choixSexe" value="idk" id="choixIdk" /> <label for="choixIdk">Je ne sais pas...</label>

Cases à cocher (multiple) : input checkbox

Les checkbox renvoient plusieurs résultats (choix multiple).
Le name sera donc sous la forme d'un tableau de résultat, d'où la syntaxe nom[] 

Cochez les langages que vous apprenez :
<input type="checkbox" name="langages[]" value="html" id="choixHtml" checked /> <label for="choixHtml">HTML</label>
<input type="checkbox" name="langages[]" value="css" id="choixCss" checked /> <label for="choixCss">CSS</label>
<input type="checkbox" name="langages[]" value="js" id="choixJs" checked /> <label for="choixJs">JS</label>
<input type="checkbox" name="langages[]" value="php" id="choixPhp" /> <label for="choixPhp">PHP</label>

Listes déroulantes : select

  • select : englobe une liste déroulante
  • option : élément de liste déroulante, choix
  • value : valeur récupérée dans le traitement
  • selected: élément sélectionné par défaut
<label for="choixSexe">Vous êtes : </label>
<select name="selectSex" id="choixSexe">
	<option value="0" selected>Sélectionnez votre sexe</option>
	<option value="homme">Un homme</option>
	<option value="femme">Une femme</option>
	<option value="idk">Je ne sais pas</option>
</select>

Ajouter un fichier (pièce jointe)

<input type="file" name="avatar" />
NB : en cas de champ file, pensez à rajouter l'attribut enctype dans la balise form

Les boutons

Envoi du formulaire : input submit

<input type="submit" value="Valider le formulaire" />

Vider le formulaire : input reset

<input type="reset" value="Vider" />

Bouton générique : input button

<input type="button" value="Cliquez sur moi!" />

Depuis HTML5 : button

<button type="submit">Valider le formulaire</button>
