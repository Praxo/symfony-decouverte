CSS - Formatage du texte

Objectifs

Donner du style à son texte :

  • color
  • text-transform/align/decoration
  • Unités de taille
  • text-indent
  • font-size/weight/style
  • letter-spacing
  • line-height

Donner du style à son texte

color : couleur du texte - Documentation

p{
	/* couleur du texte rouge */
	color: red;
}

text-transform : casse du texte - Documentation

p{
	/* valeurs possibles : uppercase, lowercase, capitalize, none */
	text-transform: uppercase;
}

text-align : alignement du texte - Documentation

p{
	/* valeurs possibles : left, right, center, justify*/
	text-align: center;
}

text-decoration : décoration/formatage du texte - Documentation

p{
	/* valeurs possibles : underline, overline, line-through, none */
	text-decoration: underline;
}

Unités de longueur / taille - Documentation

Il existe différentes unités de taille :

  • px : taille absolue, "correspond au pixel écran"
  • em : taille relative, par rapport à la taille héritée
  • % : taille relative, par rapport à la taille héritée

Et ça continue !

text-indent - espace d'indentation de la première ligne - Documentation

p{
	/* valeurs possibles : px, em, %... */
	text-indent: 2em;
}

font-size : taille de la police - Documentation

p{
	/* valeurs possibles : small, medium, large, px, em... */
	font-size: 12px;
}

p strong{
	/* 1,5 fois la valeur de la valeur de la taille héritée = 150% */
	font-size: 1.5em; 
}

font-weight : graisse de la police - Documentation

p{
	/* valeurs possibles : normal, bold, bolder, lighter, 100-900... */
	font-weight: bold;
}

font-style : change l'aspect de la police - Documentation

p{
	/* valeurs possibles : normal, italic, oblique */
	font-style: italic;
}

letter-spacing : espace entre les caractères - Documentation

p{
	/* valeurs possibles : normal, px, em... */
	letter-spacing: 1.6px;
}

line-height : hauteur de la ligne - Documentation

p{
	/* valeurs possibles : normal, px, em, %... */
	line-height: 22px;
}
