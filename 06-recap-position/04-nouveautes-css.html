Reset CSS

Comme vous avez pu le voir, certains éléments HTML ont des comportements par défaut.
Le reset CSS (ou cleaner) est un outil qui remet à 0 la valeur de certains éléments HTML afin d'éviter une partie des différences d'affichage sur les navigateurs (marges, graisses, tailles de texte, hauteur de ligne...).
Il s'agit d'un fichier CSS à inclure dans votre projet: Téléchargez le ici 

<link rel="stylesheet" type="text/css" href="styles/reset.css" /> <!-- reset CSS -->
<link rel="stylesheet" type="text/css" href="styles/style.css" /> <!-- Votre style CSS -->

box-sizing - Documentation

Par défaut les block fonctionnent comme ça : width + padding + border = largeur du block.
box-sizing permet de modifier le fonctionnement des modèles de boîtes qui est utilisé pour calculer la largeur et la hauteur des éléments. 

p{
	width: 500px;
	height: 150px;
	padding: 20px;
	border: 5px solid #000;

	box-sizing: border-box; /* width et height = contenu + padding + border */
}

Par habitude, on va toujours utiliser cette propriété pour toutes les balises !
Le sélecteur * permet de sélectionner toutes les balises. 

*{
	box-sizing: border-box;
}

Les ombres

Ombres de boites - Documentation

Valeurs nécessaires :
  • longueur longueur : décalages X et Y
  • longueur : rayon de l'ombre
  • Couleur : couleur de l'ombre (nom, hexa, rgb, rgba...)
  • Interne : inset (facultatif)
/* ombre interne */
.boite-ombre{
	box-shadow: 10px 5px 5px black inset;
}

Ombres de texte - Documentation

Les valeurs sont exactement les mêmes

.texte-ombre{
	text-shadow: 1px 1px 2px black;
}

Polices d'icônes - Font awesome - Font awesome

Des polices d'icônes sont disponibles sur le web (font awesome, glyphicon, icomoon, flaticon...)
Il suffit d'intégrer le fichier CSS de la police, et d'utiliser les class prévues à cet effet. 

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/font-awesome/4.5.0/css/font-awesome.min.css">
....
<i class="fa fa-comment"></i>
