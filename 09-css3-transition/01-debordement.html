Overflow - Documentation

overflow permet de gérer les débordements dans une boîte.

  • visible : non rogné (défaut)
  • hidden : rogné et scroll cachés
  • scroll : rogné et force l'affichage des scroll
  • auto : rogné et affiche si le contenu déborde
.zone-scroll {
	overflow: scroll; /* force l'affichage des scroll et le contenu est rogné */
}
On peut également définir : overflow-x et overflow-y

word-wrap - Documentation

word-wrap permet de gérer la césure (retour ligne) à l'intérieur d'un mot, si celui-ci est trop long et dépasse de la boîte.

  • normal
  • break-word : si pas de césure normal possible, on force le retour ligne et le mot sera coupé
.contenu-coupe{
	word-wrap: break-word;
}

white-space - Documentation

white-space formate les textes dans une boîte : comment les espaces vont être géré (retour ligne ou non).

  • normal
  • nowrap : pas de retour ligne
.ligne{
	white-space: nowrap;
}
