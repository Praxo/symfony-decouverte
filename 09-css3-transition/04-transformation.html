Transformations

Depuis CSS3, on peut apporter des transformations avec la propriété transform et une liste de fonctions prédéfinies.

Origine de la transformation

transform-origin permet de définir l'origine de la transformation. Par défaut, il s'agit du centre de l'élément.

Valeurs possibles : top, bottom, left, right, center, px, em, %...

.power-rangers{
	transform-origin: 50% 50%; /* X, Y */
	transform-origin: top left;
}

Rotation

.power-rangers{
	transform: rotate(30deg); /* valeur négative possible */
}

Changement d'échelle

.power-rangers{
	transform: scale(2, 0.5); /* X, Y : nombre de fois la valeur initiale */
}

Horizontal

.power-rangers{
	transform: scaleX(2);
}

Vertical

.power-rangers{
	transform: scaleY(0.5);
}

Inclinaison

.power-rangers{
	transform: skewX(20deg) skewY(15deg);
}

Translation

.power-rangers{
	transform: translateX(-50%) translateY(50px);
}
