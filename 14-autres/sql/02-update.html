Modifier des données en SQL

Pour modifier des données en SQL, on utilise la requête UPDATE

UPDATE post SET titre='post 3 modifié', contenu='Contenu du post 3 modifié' WHERE id=3

Via SQL & PHP/PDO

$db->exec('UPDATE post SET titre=\'post 3 modifié\', contenu=\'Contenu du post 3 modifié\' WHERE id=3');

Idem que pour l'insertion, utilisez $db->quote($chaine) pour préparer vos chaines.
