Déclarer le CSS

Inline - directement dans une balise HTML via l'attribut style - à proscrire !

<p style="..."></p>

Balise style - directement dans le head de l'HTML

<style>...</style>

Fichier externe (à privilégier) - dans la balise head - à privilégier !

<link type="text/css" rel="stylesheet" href="" />
href aura comme valeur, le chemin vers le fichier CSS

Sélecteurs / déclaration

Élément - sélectionne les balises du type

p{...}

<p> </p>

Class - sélectionne les balises ayant cet attribut class (à privilégier)

Le caractère correspondant à une class est . (point).
On peut utiliser autant de fois une class, et on peut ajouter plusieurs class à une balise. 

.header{...}

<p class="header"> </p>

ID - sélectionne la baliste ayant cet attribut id

Le caractère correspondant à un id est # (dièse -et non hashtag didiou !-)
Un id est unique et on ne peut avoir qu'un seul id. 

#main{...}

<p id="main"> </p>
