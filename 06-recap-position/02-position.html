Retour sur le float

float (left, right, none)

Le float permet de sortir un élément du flux normal, en le plaçant tout à gauche ou tout à droite de son conteneur.

Rétablir le flux

Pour rétablir le flux après les éléments en float, on peut ajouter la propriété overflow: hidden au conteneur des éléments flottants, ou ajouter un block avec la propriété clear: both après les float. 
<div class="conteneur">
	<div class="colonne-gauche">
	    Lorem ipsum dolor sit amet, consectetur adipisicing elit
	</div>
	<div class="colonne-droite">
	    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Expedita delectus eius quae magnam, deleniti a aliquid !
	</div>
	<div class="cleafix"></div>
</div>

.colonne-gauche{
	float: left;
}

.colonne-droite{
	float: right;
}

/* overflow sur le conteneur OU clear après les colonnes */
.conteneur{
	overflow: hidden;
}

.cleafix{
	clear: both;
}

Marges négatives

Il est possible de définir des marges négatives à un élément, pour ajuster sa position.
.marge-negative{
	margin-top: -20px; /* remonte le bloc de 20px */
	margin-left: -40px; /* recule le bloc de 40px vers la gauche */
	margin-right: -100px; /* recule le bloc de 100px vers la droite */
}

Propriété CSS : position - Documentation

Positionnement statique : static

Valeur par défaut, les éléments s'affichent dans l'ordre, comme dans le document html.

Positionnement relatif : relative

Permet de décaler l'élément par rapport à sa position normale.

Positionnement fixe : fixed

Positionne un élément n'importe où, par rapport à la fenêtre du navigateur (l'élément reste toujours visible).
Attention : Un élément fixed sort du flux

Positionnement absolu : absolute

Positionne un élément n'importe où sur la page, par rapport à son premier parent positionné.
Attention : Un élément absolute sort du flux

Positionner son élément

Pour positionner son élément dans la fenêtre, on utilise les propriétés left, right, top, bottom.
.bloc-absolu{
	position: absolute;
	right: 0; /* 0 de la droite */
	bottom: 20px; /* 20px du bas */
	z-index: 1; /* niveau 1 */
}

.bloc-fixe{
	position: fixed;
	left: 10px; /* 10px de la gauche */
	top: 0; /* 0 du haut */
}

.bloc-relatif{
	position: relative;
}

Superposition d'éléments - z-index

Lorsque des éléments sont positionnés, il est possible de gérer l'ordre de superposition (qui sera au dessus, qui sera en dessous...)
Pour ça, on utilise la propriété z-index. Plus la valeur de z-index est élevée, plus l'élément sera au dessus (oui, je parle bien la France...). 

.bloc1{
	position: absolute; 
	z-index: 1;
}

.bloc2{
	position: absolute; 
	z-index: 2; /* S'il y a superposition avec .bloc1, il sera au dessus */
}
