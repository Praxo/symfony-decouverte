Bordures

Objectifs

Ajouter des bordures à des éléments :

  • Propriété border
  • Bordures arrondies

Border - Documentation

3 éléments sont nécessaires pour créer une bordure : sa taille (width), son style (style) et sa couleur (color).

border: width style color
border-top / right / bottom / left 

.bloc-bordure{
	border-width: 5px; /* px, em, % */
	border-style: solid; /* Valeurs possibles : solid, dotted, dashed, none,... */
	border-color: #CCC; /* Valeurs possibles : nom, rgb, hexa */
}

/* raccourci */
.bloc-bordure{
	border: 5px solid #CCC;
}

/* Uniquement la bordure d'un coté */
.bordure-gauche{
	border-left: 3px solid red;
}

Bordures arrondies - Documentation

border-radius permet d'arrondir les angles
border-top-left-radius / border-bottom-right-radius... 

.bloc-bordure{
	border-radius: 10px; /* les 4 coins */
	border-radius: 10px 5px 20px 0; /* TL:10px / TR:5px / BR:20px / BL:0 */
}
