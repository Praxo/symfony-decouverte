Modèles de boites v2

Objectifs

Aller plus loin avec les modèles de boites :

  • Block
  • Inline
  • Inline-block
  • None

Block

Les boites de type block créé automatiquement un retour ligne avant et après.
On peut modifier les dimensions (largeur, hauteur), les marges (externes, internes). Par défaut, un block prendra toute la largeur possible. 

p{
	display: block; /* p est une balise block */
}
Hn, p, div... sont des modèles de type block

Inline

Un modèle de boite inline suit le flux du contenu.
La taille est automatique en fonction du contenu. Il est possible de lui assigner des marges (externes, internes) mais uniquement horizontales. 

a{
	display: inline; /* a est une balise inline */
}
a, span, strong, em... sont des modèles de type inline

Inline-Block

Là, c'est limite de la magie !
L'inline-block se comporte comme les balises inline (reste dans le flux du contenu), mais on peut lui assigner les propriétés d'un block (width, height, padding, margin). 

li{
	display: inline-block;
}

Et ce n'est pas tout...
Les éléments en inline-block peuvent aligner verticalement les uns par rapport aux autres ! 

li{
	display: inline-block;
	vertcal-align: middle; /* top, middle, bottom, baseline */
}

None

La propriété display: none; permet de cacher un élement. 

.hidden-tab{
	display: none;
}
