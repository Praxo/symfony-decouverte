Balise - video

Attributs

  • poster="image.jpg" : vignette, image à placer lorsque la vidéo n'est pas encore lancée (défaut : première image de la vidéo)
  • controls : ajoute les boutons de controle
  • width : ...
  • height : ...
  • loop : joue la vidéo en boucle
  • autoplay : lecture automatique

Différentes sources

Les navigateurs ne reconnaisssent pas les memes formats de vidéos, du coup on peut définir différents fichiers avec la balise source :

  • H.264/MP4 : IE + Chrome (4-16) + Safari 5.0
  • ogv : FF 3.5 + Opera 10.5 + Chrome 4.0
  • webm : FF 4.0 + Opera 10.6 + IE (si codecs installés) + Chrome 6.0
<video controls autoplay>
	<source src="video.mp4" />
	<source src="video.webm" />
	<source src="video.ogv" />
	Votre navigateur ne supporte pas la balise vidéo.
</video>

Balise - audio

Attributs

  • controls : ajoute les boutons de controle
  • width : ...
  • loop : joue la vidéo en boucle
  • autoplay : lecture automatique

Différentes sources

Idem que vidéo, on va utiliser différentes sources car les navigateurs sont capricieux... :

  • mp3 : IE + Chrome + Safari
  • ogg : Chrome + FF + Opera
<audio controls autoplay loop>
	<source src="piano_mp3.mp3" />
	<source src="piano_ogg.ogg" />
	Votre navigateur ne supporte pas la balise audio.
</audio>
