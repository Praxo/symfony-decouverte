Rappel - font-family

Pour utiliser des polices différentes, on utilise la propriété font-family.

body{
	font-family: Arial;
}

Les polices sont installées sur l'ordinateur des visiteurs.
Ainsi, pour qu'une police fonctionne sur le web, elle doit exister chez le client (on ne peut pas forcer l'installation d'une police), sinon le navigateur en prendra une par défaut.
Il est donc préférable de donner une liste de police pour proposer différentes polices (notion de family) :
si la première n'existe pas, on prend la seconde, etc...

	font-family: Courier, "MS Courier New", Prestige, monospace;

Google web font

Google met à disposition différentes polices pour le web : Google webfonts
Il suffit de les importer comme un fichier CSS.

<link href='https://fonts.googleapis.com/css?family=Lato:400,700' rel='stylesheet' type='text/css'>

body{
	font-family: "Lato", Arial, Verdana;
}

Police personnalisée : @font-face - Documentation

On peut utiliser (presque) n'importe quelle police grâce à @font-face.
Attention cependant : au poids des polices (elles devront être téléchargées par le navigateur), aux droits d'auteur et aux surtout, aux formats.
En effet, il faut différents formats pour que ça fonctionne sur tous les navigateurs : ttf (tous et IE9+), eot (uniquement IE), otf (tous sauf IE), svg (iphone, ipad), woff (tous et IE9+). 

@font-face {
    font-family: 'maSuperFont';
    src: url('maSuperFont-regular.eot') format('eot'),
         url('maSuperFont-regular.woff') format('woff'),
         url('maSuperFont-regular.ttf') format('truetype'),
         url('maSuperFont-regular.svg') format('svg');
    font-weight: normal;
    font-style: normal;
}

@font-face {
    font-family: 'maSuperFont';
    src: url('maSuperFont-bold.eot') format('eot'),
         url('maSuperFont-bold.woff') format('woff'),
         url('maSuperFont-bold.ttf') format('truetype'),
         url('maSuperFont-bold.svg') format('svg');
    font-weight: bold;
    font-style: normal;
}

@font-face {
    font-family: 'maSuperFont';
    src: url('maSuperFont-bold-italic.eot') format('eot'),
         url('maSuperFont-bold-italic.woff') format('woff'),
         url('maSuperFont-bold-italic.ttf') format('truetype'),
         url('maSuperFont-bold-italic.svg') format('svg');
    font-weight: bold;
    font-style: italic;
}

body{
	/* Appelle la police maSuperFont en normal */
	font-family: 'maSuperFont';
}

h1{
	/* Appelle la police maSuperFont en bold */
	font-family: 'maSuperFont';
	font-weight: bold;
}
Note : on peut trouver différentes polices accessibles sur fontsquirrel.
