Opacités

rgba - Documentation

.elem{
	color: rgba(255, 0, 0, 0.6); /* opacité sur la couleur du texte */
	background: rgba(150, 0, 144, 0.6); /* opacité sur la couleur du background */
}

opacity - Documentation

opacity permet de gérer l'opacité d'un élément (contrairement à rgba qui gère l'opacité de la couleur).
La propriété prend une valeur comprise entre 0 (transparent) et 1 (opaque). 

.elem{
	opacity: 0.6;
}

Visibilité - Documentation

La propriété visibility peut être utilisée pour cacher un élément, tout en laissant l'espace qu'il aurait occupé 

.elem{
	visibility: hidden; /* visible, hidden, ... */
}
