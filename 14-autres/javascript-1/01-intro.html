Différences entre les langages

  • HTML : Langage structurel d'une page
  • CSS : Mise en forme
  • javascript : Gère / ajoute du comportement à une page (langage client -> navigateur)
  • PHP : Langage serveur, traite la demande, relation BDD et renvoi un résultat

Chaque langage est complémentaire mais empiète plus ou moins également sur les autres (exemple : :hover est un comportement CSS qui empiète sur le JS).

Protocole HTTP

Pour chaque ressource dont le navigateur a besoin (HTML, fichiers CSS, scripts, images...), une requête HTTP est effectuée à une adresse précise : l'URL.

Le navigateur web (Client) envoie une requête HTTP au serveur.
Le serveur traite la demande et renvoie une réponse HTTP au client, avec un code réponse (200 OK, 404 non trouvé, 500 erreur, 301 redirection...)

.

Ces "conversations" client - serveur sont visibles dans l'onglet "Réseau" ou "Network" de votre inspecteur navigateur.

Utilisation du Javascript

Comme le CSS, il est possible de définir du javascript directement dans le fichier HTML, ou de faire appel à un fichier externe.

Directement dans le fichier HTML

<script type="text/javascript">
	window.alert("test");
</script>

Appel d'un fichier externe

<script type="text/javascript" src="./scripts/fichier.js"></script>
