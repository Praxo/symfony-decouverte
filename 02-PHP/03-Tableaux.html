HTML - Liens et URLs

Objectifs

Créer un lien hypertexte :

  • Balise de lien
  • "Cadre" de lien
  • Infobulle au survol du lien
  • Lien interne - autre page de son site
  • Lien vers une ancre

Documentations / Liens

  • Openclassrooms : Créer des liens W3school : Targets

Balise de lien

<a href="http://www.google.com/">Lien vers Google</a>
  • a : Balise html
  • href : Attribut obligatoire de la balise <a>, définit la destination
  • http://www.google.com/ : Valeur de l'attribut href
  • "Lien vers Google" : Label du lien (texte sur lequel on clique)
Pour un lien externe, il faut préciser le protocole

"Cadre" de lien

Il est possible de définir le cadre d'ouverture du lien (fenêtre courante (par défaut), nouvelle fenêtre, ...)

<a href="http://www.google.com/" target="_blank">Lien vers Google</a>
  • target="_blank" : Le lien s'ouvre dans une nouvelle fenêtre

Exemple

Google

Infobulle au survol du lien

<a href="http://www.google.com/" title="Cliquez ici">Lien vers Google</a>
  • title="Cliquez ici" : Attribut définissant un titre de lien - valeur textuelle uniquement

Exemple

Google

Lien interne - autre page de son site

Le fichier de destination se situe au même niveau

<a href="page2.html">Lien vers la page 2</a>

Le fichier de destination se situe dans un sous-dossier "contenu"

<a href="contenu/page2.html">Lien vers la page 2 </a>

Le fichier de destination se situe dans un dossier parent

<a href="../page2.html">Lien vers la page 2</a>

Naviguer dans l'arborescence

  • . : à partir du dossier courant
  • .. : revient vers au dossier parent
  • / : à partir du dossier racine

Lien vers une ancre

Une ancre est un repère dans la page courante (Exemple)
Pour créer une ancre, il suffit d'ajouter l'attribut id à n'importe quelle balise html. 

<h3 id="biographie">Ma biographie</h3>

Pour créer un lien vers l'ancre, on utilise le caractère # (symbole de l'id) 

<a href="#biographie">Aller vers l'ancre biographie</a>
L'attribut id doit être unique dans un document HTML.
