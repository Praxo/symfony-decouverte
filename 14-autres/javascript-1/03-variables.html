Une variable, une constante c'est quoi ?

Variable

Une variable est un emplacement dans la mémoire vive de l'ordinateur pour stocker de l'information.
Elle possède un nom, un type et une valeur.
La valeur d'une variable peut évoluer et être modifiée.

Constante

Une constante est -quasi- similaire à une variable, sauf qu'on ne peut plus changer sa valeur après sa déclaration (lecture seule).

Types de variables

Le type d'une variable permet au langage de savoir comment manipuler la valeur de la variable.
Pour le moment, on se limitera aux types :

  • String : Chaine de caractère, encadré par des ""
  • Int (ou Integer) : Valeurs numériques entières, sans ""

Déclarer une variable

Le mot clé var permet de déclarer une variable.
Il faut toujours déclarer une variable.
Attention au nommage des variables : évitez les caractères spéciaux, préférez le camelCase ou la séparation avec un _.

var age = 10;
  • var : mot clé pour déclarer la variable
  • age : nom donné à la variable (attention au nommage : camelCase, ou séparé par des "_"
  • 10 : valeur de la variable
  • Cette variable est de type Integer (nombre)
var age = "10";

Cette variable est de type String (chaine de caractères).

Utiliser les variables

Lorsqu'une variable est déclarée, on peut la modifier.

var age = 10; // age vaut 10 et a un type int
age = 20; // age vaut 20
var test = age; // test vaut age (donc 20) et a un type int
var prenom = "Roger"; // prenom est une String et a pour valeur Roger
document.write(age); // affiche 20 sur la page
window.alert(prenom); // affiche une alerte contenant la valeur de prenom, donc Roger

Déclarer une constante

Le mot clé const permet de déclarer une constante.
Par convention, les constantes seront nommées en majuscule.
Pour rappel, une fois déclarée, une constante ne peut plus changer de valeur.

const PI = 3.14159265359;

Opérateurs arithmétiques

Opérateurs arithmétiques de bases

+, -, *, /, %

var age = 10;
age = age + 1; // age vaut 11

Raccourcis pour les opérations arithmétiques sur une même variable

+=, -=, *=, /=, %=, ++, --

var age = 10;
age += 1; // correspond à : age = age + 1
age++; // correspond à : age = age + 1
