Propriété - color

a{
	color: red;
}

Syntaxes des couleurs

Nominative

a{
	color: red; /* Valeurs possibles : red, blue, white, black, yellow,... */
}

Syntaxe RGB

Permet de définir les composants de Red (rouge), Green (vert) et Blue (bleu), avec des valeurs allant de 0 (0%) à 255 (100%).
RGB = RVB = Rouge Vert Bleu.

span{
	color: rgb(0, 255, 0); /* vert, peut être noté : rgb(0%, 100%, 0%) */
	color: rgb(0, 0, 0); /* noir (aucune couleur) */
	color: rgb(255, 255, 255); /* blanc (toutes les couleurs), peut être noté : rgb(100%, 100%, 100%) */
}

Hexadécimale

Syntaxe : #RRGGBB. Les composantes rouges, vertes et bleues sont fixées sur un interval de 0-9 et A-F (A=10, F=15). 
span{
	color: #000; /* Correspond à #000000, black, rgb(0, 0, 0) */
	color: #ff0000; /* Correspond à #f00, red, rgb(255, 0, 0) */
	color: #ffffff; /* Correspond à #fff, white, rgb(255, 255, 255) */
}
