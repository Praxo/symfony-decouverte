Conteneur

Une grille est un ensemble de lignes, découpées en cellules.
La grille de bootstrap est divisée en 12 colonnes fluides (en %). Une ligne sera donc "découpée" en 12 colonnes.

Système de grille - Documentation

La grille Bootstrap doit être placée dans un conteneur.

  • container : centre la grille sur une largeur fixe qui s'adapte
  • container-fluid : prend toute la largeur (100%)

Le conteneur est responsive, ainsi, il s'adaptera toujours :

Type d'écran Grille Taille d'écran Taille du container
Smartphone xs : extra small devices < 768 pixels auto
Tablette sm : small devices >= 768 pixels 750px
Écran de bureau md : medium devices >= 992 pixels 970px
Grand écran de bureau lg : large devices >= 1200 pixels 1170px 
<div class="container">
	Conteneur de ma grille
</div>

Organisation de la grille - ligne

Une "ligne" est définie par la class row. La class row sert entre autre de clearfix. 

<div class="container">
	<div class="row">
		Contenu de ma ligne
	</div>
</div>

Organisation de la grille - Colonnes

Pour définir une colonne, il faut lui donner une dimension. La grille étant divisée en 12 colonnes :

  • 1 : 1/12 = 8.33%
  • 4 : 4/12 = 33.33%
  • 6 : 6/12 = 50%
  • ...

Bootstrap est reponsive, chaque class de grille correspond en fait à un media query, sous la forme col-[device]-[nb col] :

Type d’écran Grille Taille d’écran Classes
Smartphone xs : extra small devices < 768px .col-xs-*
Tablette sm : small devices >= 768px .col-sm-*
Écran de bureau md : medium devices >= 992px .col-md-*
Grand écran de bureau lg : large devices >= 1200px .col-lg-*

Ainsi, il est possible de combiner plusieurs classes à un élément, et gérer son comportement dans les différentes résolutions : 

<div class="container">
	<div class="row">
		<div class="col-lg-3 col-sm-6">Largeur de 3</div> <!-- 3/12 quand >=1200 / 6/12 quand >=992 -->
		<div class="col-lg-1 col-sm-3">Largeur de 1</div> <!-- 1/12 quand >=1200 / 3/12 quand >=992 -->
		<div class="col-lg-8 col-sm-3">Largeur de 8</div> <!-- 8/12 quand >=1200 / 3/12 quand >=992 -->
	</div>
</div>

Exercice

Réaliser le schéma suivant

Seules propriétés css autorisées :

body{
	background: #f2f2f2;
}

/* Espace en dessous des lignes */
.row{
	margin-bottom: 20px;
}

/* Styles pour les colonnes */
[class^="col-"]{
	background:#fff;
	border: 1px solid #CCC;
}

/* ET C'EST TOUT ! */

Seules propriétés css autorisées :

body { margin-top: 10px; }
img { width: 100%; }
[class*="col"] { margin-bottom: 20px; }

Décaler les éléments -colonnes- d'une grille

Sur le même principe, il est possible de décaler des colonnes grâce à une série de class col-[device]-offset-[nb col] (se traduit par un margin-left)

Type d’écran Grille Taille d’écran Classes
Smartphone xs : extra small devices < 768px .col-xs-offset-*
Tablette sm : small devices >= 768px .col-sm-offset-*
Écran de bureau md : medium devices >= 992px .col-md-offset-*
Grand écran de bureau lg : large devices >= 1200px .col-lg-offset-* 
<div class="container">
	<div class="row">
		<div class="col-md-6">Largeur de 6</div>
		<div class="col-md-4 col-md-offset-2">Largeur de 4 avec décalage de 2</div>
	</div>
</div>
