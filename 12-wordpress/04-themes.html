Les templates - Documentation

Un template est un fichier PHP appelé par Wordpress pour générer le contenu HTML.
Lorsque vous êtes sur une page, Wordpress recherche le type de cette page (listing des posts, page statique, article, page d'accueil...). En fonction de la page courante, il appelera un fichier/template différent : Schéma des appels de templates

Pour qu'un thème soit valide, il faut obligatoirement qu'il contienne un fichier de template : index.php, qui est le fichier appelé par défaut.
