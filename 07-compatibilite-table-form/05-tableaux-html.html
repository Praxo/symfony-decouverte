Tableaux HTML

Objectifs

Construire un tableau html :

  • Balises de bases : table, tr, td
  • Balises de structure : caption, thead, tbody, tfoot, th
  • Fusion des cellules
  • Propriétés CSS tableaux

Création d'un tableau basique

  • table : déclarer un tableau
  • tr : ligne de tableau
  • td : cellule de tableau
<table>
	<tr>
		<td>cellule 1 / ligne 1</td>
		<td>cellule 2 / ligne 1</td>
	</tr>
	<tr>
		<td>cellule 1 / ligne 2</td>
		<td>cellule 2 / ligne 2</td>
	</tr>
</table>

Tableau structuré

  • caption : titre du tableau
  • thead : header du tableau
  • tbody : corps du tableau
  • tfoot : pied du tableau
  • th : cellules mises en avant (par défaut en gras, remplace td dans thead ou dans tfoot)
<table>
	<caption>Titre du tableau</caption>
	<thead>
		<tr>
			<th>Entete 1</th>
			<th>Entete 2</th>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>c1 l1</td>
			<td>c2 l1</td>
		</tr>
		<tr>
			<td>c1 l2</td>
			<td>c2 l2</td>
		</tr>
	</tbody>
	<tfoot>
		<tr>
			<th>Pied 1</th>
			<th>Pied 2</th>
		</tr>
	</tfoot>
</table>

Fusion des cellules

  • colspan : nombre de colonnes que doit faire une cellule : fusion horizontale (1 par défaut)
  • rowspan : nombre de lignes que doit faire une cellule : fusion verticale (1 par défaut)
<table border="1">
	<tr>
		<td colspan="2">2 colonnes (fusion horizontale)</td>
	</tr>
	<tr>
		<td rowspan="2">2 lignes (fusion verticale)</td>
		<td>cellule 2 / ligne 2</td>
	</tr>
	<tr>
		<td>cellule 2 / ligne 3</td>
	</tr>
</table>

Donner du style à un tableau

table{
	border-collapse: collapse; /* bordures confondues : collapse, separate */
}

td{
	vertical-align: middle; /* alignement vertical : top, left, middle, bottom */
}
