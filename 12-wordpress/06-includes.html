Includes de fichiers

Certains contenus (codes HTML / PHP) seront identiques pour tous les templates. On peut donc les regrouper dans différents fichiers et les inclure.

  • header.php : début du code (doctype, head, body ouvrant)
  • footer.php : fin du code (fermeture body, html)

header.php / footer.php

Le fichier header.php est celui, avec footer.php, que l'on place dans tous les templates. L'un ouvre le code HTML, contient le doctype, le head, le body, l'autre le ferme (fichiers JS, fermeture du body & html).
Le template se chargera des contenus.

  • header.php : début du code (doctype, head, body ouvrant)
  • footer.php : fin du code (fermeture body, html)
<!DOCTYPE html>
<html>
<head <?php language_attributes(); ?>>
    <meta charset="<?php bloginfo('charset'); ?>">
    <link rel="stylesheet" href="<?php bloginfo('stylesheet_url'); ?>" type="text/css">
    <?php wp_head(); ?>
</head>
<body>
    <div class="container">
        <header>
            <h1><a><?php bloginfo('name'); ?></a></h1>
            <h2><?php bloginfo('description'); ?></h2>
        </header>
        <main>

        </main>
        <footer>
            <?php bloginfo('name'); ?> est propulsé par <a href="http://wordpress.org">WordPress</a>.
        </footer>
    </div>
    <?php wp_footer(); ?>
</body>
</html>
  • wp_head() : permet aux plugins d'insérer du code dans le head
  • wp_footer() : idem wp_head (notamment les appels JS)

Inclure les fichiers dans les templates

Maintenant que nous avons définis les includes de base (header et footer), utilisons-les dans un template : index.php.

<?php
	get_header(); // inclut header.php
?>

	<!-- Contenu du template -->
	Contenu du template index.php.

<?php
	get_footer(); // inclut footer.php
?>
