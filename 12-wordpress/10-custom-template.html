Pourquoi créer un template ?

Toutes les pages n'auront pas forcément la meme apparence...
Une des particularités des Pages, est de pouvoir créer des modèles de Page, différent du template page.php.

Un modèle de page est un fichier template (non natif de Wordpress), avec une boucle, et peut être assigné à une page particulière.

Création d'un modèle de Page

Pour créer un modèle de page, il suffit de créer un fichier php, et d'ajouter un commentaire en début de fichier : 

<?php
/*
	Template Name: Contact
*/
?>

La suite est une boucle habituelle avec la structure HTML que vous souhaitez...

NB : J'ai pour habitude de préfixer les modèles de page par tpl- (ex : tpl-contact.php, tpl-portfolio.php), ça permet de les distinguer des templates de base.

Assigner un modèle à une page

Une fois le modèle créé, il faut l'assigner à une page : Gestion de la page souhaitée > Attributs de la page (colonne de droite) > Modèle
