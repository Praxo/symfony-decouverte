Environnement de travail

  • Système d'exploitation : Ubuntu - basé sur GNU/Linux, OS Open source, libre et gratuit
  • Navigateurs : Firefox et/ou Google Chrome
  • Éditeur de texte : Sublime Text 2
  • Outil : Firebug / inspecteur : outil de développement / debug
  • Serveur apache (localhost) : serveur permettant d'interpréter le PHP et de gérer les BDD

Internet

Mise en réseau de machines pour communiquer entre elles.

Quelques dates :

  • 1990 : Première version de l'HTML

  • 1997 : Création du xHTML
  • 1998 : Mise en forme CSS pour séparer le fond de la forme
  • 2007 : xHTML2 est abandonné au profit de l'HTML5
