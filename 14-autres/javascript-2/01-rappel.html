Utiliser du javascript

Directement dans le fichier HTML

<script type="text/javascript">
	window.alert("test");
</script>

Appel d'un fichier externe

<script type="text/javascript" src="./scripts/fichier.js"></script>

Afficher du contenu

document.write('<p>Bonjour tout le monde !</p>'); // affiche du contenu dans la page
console.log('hello console !'); // affiche du contenu dans la console de développement (F12)
window.alert('hello popup !'); // ouvre une fenêtre modale
window.confirm("Valider la demande ?"); // modale avec 2 boutons : OK / Annuler
window.prompt("Saisissez votre nom."); // modale avec champ texte

Variables, constantes, opérateurs

Une variable stocke une information, elle a un nom, un type (entier, string, objet...) et une valeur.

Une constante, c'est pareil, sauf qu'elle n'évoluera pas.

var age = 10; // variable "nom" - "entier" qui vaut 10
var age = "10"; // variable "nom" - "string" qui vaut 10
const TVA = 19.6; // constante

var age = 10;
age = age + 1;
age += 1; // idem
age++; // idem

Concaténation

var age = 26;
document.write("Bonjour, j'ai " + age + " ans");

// forcer javascript à faire une addition
var age = "10";
var toto = "20.5";
var total = parseInt(age) + parseFloat(toto);
document.write(total); // affichera : 30.5

Les tableaux de données

var tableau = new Array(); // déclaration d'une variable tableau de type objet de la classe Array
tableau[0] = "rouge"; // indice : 0 / valeur : rouge
tableau[1] = "vert"; // indice : 1 / valeur : vert
tableau[2] = "bleu"; // indice : 2 / valeur : bleu

// correspond aussi à : var tableau = ["rouge", "vert", "bleu"];

alert(tableau[2]); // affiche bleu
Un tableau commence avec clé numérique toujours à la clé 0

Tableaux associatifs

var user = new Array();
user["nom"] = "Depardieu";
user["prenom"] = "Gérard";
user["hobby"] = "Vin rouge";

document.write(user["nom"]); // affiche l'élément à la clé "nom", donc Depardieu
