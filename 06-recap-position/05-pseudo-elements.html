Pseudo classe - Documentation

Une pseudo-classe est une manière de cibler un élément sans ajouter une classe.

  • :first-child : premier enfant
  • :last-child : dernier enfant
  • :not() : élément qui n'est pas...
  • :nth-child() : sélectionne les éléments en fonction de leur "index"
    • :nth-child(odd) : les éléments fils impairs (1e, 3e, 5e...)
    • :nth-child(event) : les éléments fils pairs (2e, 4e, 6e...)
    • :nth-child(3) : le 3e élément
  • Liste des pseudos classes
ul li:first-child{ /* premier enfant li de ul */
	background: red;
}
p:not(.intro){ /* p qui n'a pas la class intro */
	font-weight: normal;
}
ul li:nth-child(even){ /* tous les enfants li avec un index pair */
	color: yellow;
}

Pseudo élément - Documentation

Un pseudo-élément est une chose que l'on n’aurait pas pu cibler autrement qu'en rajoutant un élément

  • :first-letter : première lettre
  • :first-line : première ligne
  • Liste des pseudos éléments
p:first-letter{ /* première lettre de p */
	font-size: 20px;
	font-weight: bold;
	text-transform: uppercase;
}

Pseudo élément magiques : :after & :before

:after et :before permettent d'ajouter du contenu inline directement après ou avant le contenu d'un élément.
content, propriété obligatoire de after/before, permet de spécifier le contenu qui est inséré. 
<h1>Mon titre principal</h1>

h1{
	color: red;
}

h1:after{
	content: " C'était un titre";
	font-size: 12px;
	color: blue;
}
h1:before{
	content: "Titre : ";
	text-transform: uppercase;
	color: green;
}

Astuce clearfix & after

Lorsqu'on utilise des float, il faut revenir dans le flux et on avait vu 2 solutions :

  • overflow: hidden sur le parent
  • clear: both sur un élément placé directement après les float

Le pseudo élément :after, ajoute un "pseudo contenu" après le contenu d'un élément. Vous voyez où je veux en venir ? 

<div class="wrapper">
	<div class="colonne"></div>
	<div class="colonne"></div>
</div>

.colonne{
	float: left;
	width: 50%;
}

.wrapper:after{
	content: "";
	display: block;
	clear: both;
}
