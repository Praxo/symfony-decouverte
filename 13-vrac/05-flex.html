Modèle de boîte flexible

Le module CSS3 Flexbox Layout fournit une façon plus efficace de disposer, aligner et distribuer l'espace entre les items d'un container, même lorsque leurs dimensions sont inconnues et/ou dynamiques - d'où le terme "flex".

Du coup, on ne raisonne plus en block / inline / float, mais en :

  • Distribution des éléments horizontale ou verticale, avec passage à la ligne autorisé ou non
  • Alignements et centrages horizontaux et verticaux, justifiés, répartis
  • Réorganisation des éléments indépendamment de l'ordre du flux (DOM)
  • Gestion des espaces disponibles (fluidité)

En résumé, on a un conteneur, et on organise les éléments enfants dans ce conteneur.

Modèle HTML

On va partir sur un modèle simple :

<div class="container">
	<div class="element">element 1</div>
	<div class="element">element 2</div>
	<div class="element">element 3</div>
</div>

Display flex

En assignant un display: flex au conteneur, les éléments internes vont se placer côte à côte (par défaut, ils sont les uns en dessous des autres).
Les éléments internes sont considérés comme flex-items.

.container{
	display: flex;
}

Distribution

Direction - flex-direction

flex-direction permet de gérer la distribution, le sens d'affichage (horizontal, vertical...).

.container{
	display: flex;
	flex-direction: row; /* row / row-reverse / column / column-reverse */
}

Gérer les retours lignes - flex-wrap

Par défaut, en HTML/CSS, lorsqu'un élément n'a plus la place de se positionner à coté, il va à la ligne.
Avec flex-wrap il est possible de définir si les éléments doivent retourner à la ligne, ou si on les force à rester sur la même ligne (leurs dimensions s'adapteront).

.container{
	display: flex;
	flex-wrap: nowrap; /* nowrap / wrap / wrap-reverse */
}

Alignements

Les éléments sont organisés soit horizontalement (row), soit verticalement (column). Cela définit l'axe principal. Il y a aussi un axe secondaire, qui correspond à l'opposé.

Aligner l'axe principal - justify-content

justify-content permet de gérer l'alignement, comment les éléments vont se présenter sur l'axe principal

  • flex-start : alignés au début
  • flex-end : alignés à la fin
  • center : alignés au centre
  • space-between : dispatchés sur tout l'axe (espaces entre chacun, pas d'espaces aux extrémités)
  • space-around : dispatchés sur tout l'axe (espaces entre chacun, espaces aux extrémités)
.container{
	display: flex;
	justify-content: flex-start; /* flex-start / flex-end / center / space-between / space-around */
}

Aligner l'axe secondaire - align-items

align-items permet de gérer l'alignement, comment les éléments vont se présenter sur l'axe secondaire

  • stretch : alignés sur tout l'axe
  • baseline : alignés sur la ligne de base
.container{
	align-items: stretch; /* stretch / flex-start / flex-end / center / baseline */
}

Aligner un élément particulier - align-self

align-self permet d'aligner un "flex-item" en particulier (pour le distinguer des autres)

.element:nth-child(2){ /* 2e bloc */
	align-self: flex-end; /* stretch / flex-start / flex-end / center / baseline */
}

Répartir sur plusieurs lignes

align-content répartit les différentes lignes flexbox sur la hauteur.

.container{
	display: flex;
	flex-wrap: wrap;
	align-content: flex-end; /* stretch / flex-start / flex-end / center / space-between / space-around */
}

Flexibilité - flex

Les flex-items prennent la taille de leur contenu (un peu comme de l'inline).

flex permet de rendre un élément flexible, en lui assignant une valeur > 0.
Cet élément occupera donc l'espace restant au sein de son conteneur.

.element:nth-child(2){
	flex: 1;
}
.element:nth-child(3){
	flex: 2; /* peut "grossir" 2 fois plus" que le second élément */
}

Ordre - order

Il est possible de modifier l'ordre des éléments avec order

.element:nth-child(1){
	order: 3;
}

.element:nth-child(2){
	order: 1;
}

.element:nth-child(3){
	order: 2;
}
