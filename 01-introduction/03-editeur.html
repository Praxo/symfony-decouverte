Prise en main du logiciel Sublime Text 2

  • Éditeur de texte ou IDE
  • Fenêtres
  • Raccourcis claviers
Sublime Text est un éditeur de texte (ne pas confondre avec un traitement de texte...) multi-plateforme (windows, iOS, Linux) proposant énormément de fonctionnalités pratiques pour les développeurs. Il est disponible "gratuitement".

Éditeur de texte ou IDE

  • Éditeur de texte : logiciel proposant par défaut des fonctionnalités simples (édition de fichiers, coloration syntaxique, indentation...) - Exemples : sublime text, brackets
  • IDE (Integrated Development Environment) : logiciel dédié au développement avec des fonctionnalités plus poussées (debug, compilation, gestion de projets plus évoluée, console intégrée...) - Exemples : phpstorm, netbeans

Durant la formation, nous utiliserons sublime text 2, qui est très intéressant et customisable (je l'utilise quotidiennement).

Fenêtres

  • Gauche : fichiers, projets ouverts dans l'éditeur
  • Droite : fichiers ouverts, zone d'édition
  • "Super" Droite : scroll rapide visuel

Raccourcis clavier (les plus utilisés)

  • CTRL + N : nouveau fichier
  • CTRL + S : sauvegarder
  • CTRL + C : copier
  • CTRL + V : coller
  • CTRL + F : rechercher
  • CTRL + MAJ + F : rechercher / remplacer
  • CTRL + G : accèder à la ligne renseignée
