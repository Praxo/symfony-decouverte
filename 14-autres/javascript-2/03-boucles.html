Les boucles

Une boucle permet de répéter des actions pour éviter de les écrire plusieurs fois...

  • Incrématations
  • Boucle While
  • Exerice While
  • Boucle For

Incrématations

var numero = 0;
numero = numero + 1;
numero += 1;
numero++;

Boucle While

Pour faire fonctionner une boucle, il faut définir une condition. Tant que la condition est vraie (true), la boucle se répète, sinon, elle s'arrête (false).

var number = 1;

while(number < 10) {
	number++;
}

document.write(number); // Affiche 10

Boucle For

Même fonctionnement que while, mais syntaxe différente. Principle d'un for :

for ( initialisation ; condition ; incrémentation ){
	// ...
}

for ( var iteration = 0; iteration < 10; iteration++) {
	document.write('Itération n° ' + iteration);
}

Afficher le contenu d'un tableau

var tableau = ["toto", "tata", "titi"]

for ( var iteration = 0; iteration < tableau.length; iteration++) {
	document.write('Ligne n°' + iteration + " : " + tableau[iteration] + "<br>");
}
