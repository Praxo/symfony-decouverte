Flux, float & clear

Objectifs

Faire "flotter" des éléments :

  • Comprendre la notion de flux
  • Sortir un élément du flux : le faire flotter
  • Annuler le comportement du float / rétablir le flux

Documentations / Liens

  • Initiation au positionnement CSS : position float

Comprendre la notion de flux

Le flux HTML peut se définir comme comme étant le comportement naturel d'affichage des éléments d'une page web.
Autrement dit, les éléments se place les uns à la suite des autre depuis le haut de la page (comme ils ont été écrit dans l'HTML).
On peut le schématiser par une flèche allant du bas vers le haut.

Faire flotter un élément - float

On peut sortir un élément du flux en lui donnant la propriété float.
L'élément peut être sorti du flux par la droite (right), par la gauche (left). 

<div class="colonne-wrapper">
	<div class="colonne-gauche">...</div>
	<div class="colonne-droite">...</div>
</div>

.colonne-gauche{
	background: red;
	width: 60%;
	height: 300px;
	float: left;
}

.colonne-droite{
	background: blue;
	width: 35%;
	height: 400px;
	float: right;
}
Pour annuler un float, on utilise la valeur none : float:none;

Rétablir le flux - clear

Les éléments en float étant hors du flux, leur hauteur n'est pas prise en compte.
Pour comptabiliser la hauteur des éléments flottants dans le flux (et ainsi revenir dans la flux normal), il faux ajouter un "clear" qui va annuler le comportement du float.
Plusieurs solutions :

Propriété - clear

<div class="colonne-wrapper">
	<div class="colonne-gauche">...</div>
	<div class="colonne-droite">...</div>
	<div class="clearfix"></div>
</div>

.clearfix{
	clear: both; /* rétablis le flux après des float */
}

Propriété - overflow

Overflow permet (entre autre, on en reparlera...) de créer un contexte de formatage, c'est à dire une "zone indépendante" qui reviendra automatiquement dans le flux.
La propriété overflow doit envelopper les float. 

<div class="colonne-wrapper">
	<div class="colonne-gauche">...</div>
	<div class="colonne-droite">...</div>
</div>

.colonne-wrapper{
	overflow: hidden; /* rétablis le flux après des float */
}

En résumé

Pour réaliser des colonnes en utilisant la propriété float, il faut:

  • Un élément parent
  • Autant d'éléments enfants que de colonnes avec:
    • la largeur (width) définie
    • la propriété float
  • un clear
