Les Tableaux en JS

Objectifs

Prise en main des tableaux en JS

  • Tableaux simples à une dimension (liste)
  • Tableaux à 2 dimensions
Array : Les tableaux JavaScript sont des objets semblables à une liste et possèdent plusieurs méthodes incorporées pour exécuter des opérations de parcours et de modification. Ni la taille d'un tableau ni les types de ses éléments n'est fixé. Puisque la dimension d'un tableau peut augmenter ou diminuer à tout moment, les tableaux ne sont pas garantis d'être compacts. En général, cela permet de couvrir plusieurs cas pratiques. En revanche si cette compacité est nécessaire pour votre cas d'utilisation, vous pourriez envisager d'utiliser les tableaux typés.
Attention : Notez que vous ne devriez pas considérer un tableau comme un tableau associatif. Vous pouvez utiliser des objets classiques à la place pour obtenir ce comportements (cette approche n'est pas sans inconvénient non plus, mais c'est la plus pertinente).

Exemple Tableau à 1 dimension

var arr = ["le premier élément", "le second élément"];
console.log(arr[0]);             // affiche "le premier élément"
console.log(arr[1]);             // affiche "le second élément"
console.log(arr[arr.length - 1]);// affiche "le second élément"
On peut parcourir un tableau en javascript directement via la méthode forEach de l'objet Array : 
var list = ["toto", "tata", "titi", "tutu", "tete"];
list.forEach(function(value)
{
	console.log(value);
});

Exemple Tableau à 2 dimensions

var plateau = [ 
  ['T','C','F','R','K','F','C','T'],
  ['P','P','P','P','P','P','P','P'],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  [' ',' ',' ',' ',' ',' ',' ',' '],
  ['p','p','p','p','p','p','p','p'],
  ['t','c','f','k','r','f','c','t'] ];

console.log(plateau.join('\n') + '\n\n');

// On déplace le pion de deux cases en avant 2
plateau[4][4] = plateau[6][4];
plateau[6][4] = ' ';
console.log(plateau.join('\n'));
