Rappels

Objectifs

Rappels :

  • Structure générale : DOCTYPE, html, head, body
  • Balises génériques : Hn, strong, em, p, br, hr, ul, ol, li
  • Liens : a href
  • Balises HTML5 : header, nav, section, article, figure, aside, footer
  • CSS : privilégier l'insertion de fichier externe : <link rel="stylesheet" href="./styles/style.css" />
  • CSS : formatage du texte : color, text-transform/align/decoration, px/em/%, text-indent, font-size/weight/style, letter-spacing, line-height
  • CSS : changer de police
  • CSS : modèles de boites : block, inline, margin, padding

Structure - HTML5

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title>Hello world</title>
		<link rel="stylesheet" type="text/css" href="styles/style.css" />
	</head>
	<body>
		<header>
			<h1>Titre principal</h1>
			<h2>Titre 2</h2>
		</header>
		<main>
			<nav>
				<ul>
					<li><a href="#">Accueil</a></li>
					<li><a href="#">Portfolio</a></li>
					<li><a href="#">Contact</a></li>
				</ul>
			</nav>

			<section>
				<article>                
					<aside>
						<p>Informations secondaires concernant l'article</p>
					</aside>
					<h1>Titre de l'article</h1>
					<p>Contenu de l'article</p>
					<figure>Illustration de l'article</figure>
					<a href="article.html">Lien vers l'article</a>
				</article>
			</section>
		</main>
		<footer>
			<p>Pied de page, date, auteur</p>
		</footer>
	</body>
</html>

CSS

Formatage du texte

p{
	color: red; /* couleur du texte */
	
	text-transform: uppercase; /* casse */
	text-align: center; /* alignement */
	text-decoration: underline; /* decoration */
	text-indent: 10px; /* indentation */

	font-weight: bold; /* graisse */
	font-style: italic; /* aspect */

	letter-spacing: 1.6em; /* espace entre les caractères */
	line-height: 2em; /* hauteur des lignes */
}

Couleurs

p{
	color: red; /* nominatif */
	color: #ff0000; /* hexadecimal */
	color: rgba(255, 0, 0); /* RGB */
}

Utiliser une police différente

@font-face{
	font-family: 'maSuperFont';
	src: url('maSuperFont-regular.eot') format('eot'),
	     url('maSuperFont-regular.woff') format('woff'),
	     url('maSuperFont-regular.ttf') format('truetype'),
	     url('maSuperFont-regular.svg') format('svg');
	font-weight: normal;
	font-style: normal;
}

.titrePrincipal{
	font-family: Arial, Helvetica, sans-serif /* penser au fallback pour les différents OS */
}

h2{
	font-family: "maSuperFont";
	font-weight: normal;
	font-style: normal;
}

Modèles de boites

p{
	display: block; /* retour ligne automatique */

	/* TAILLE */
	width: 200px; /* largeur du bloc */
	height: 400px; /* hauteur du bloc */

	/* MARGES : -top, -right, -bottom, -left */
	margin: 20px; /* marges externes */
	padding: 20px; /* marges internes */
}

a{
	display: inline; /* suit le flux de contenu */
}

/* centrer un block */
.container{
	width: 200px;
	margin: 0 auto;
}

Background

body{
	background-color: red; /* #F00, rgba(255, 0, 0); */
	background-image: url("chemin/vers/monimage.jpg");
	background-repeat: repeat; /* no-repeat, repeat-x, repeat-y... */
	background-position: right top; /* top, left, right, bottom, center, %, px... */

	/* Raccourci */
	background: red url("chemin/vers/monimage.jpg") no-repeat right top;
}
