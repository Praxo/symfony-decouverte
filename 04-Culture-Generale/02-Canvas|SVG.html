Modèles de boites

Objectifs

Introduction aux modèles de boites :

  • Modèles de boites : block, inline
  • Largeur et hauteur : width, height
  • Gestion des marges : margin, padding

Block

Les boites de type block créé automatiquement un retour ligne avant et après.
On peut modifier les dimensions (largeur, hauteur), les marges (externes, internes). Par défaut, un block prendra toute la largeur possible. 

p{
	display: block; /* p est une balise block */
}
Hn, p, div... sont des modèles de type block

Inline

Un modèle de boite inline suit le flux du contenu.
La taille est automatique en fonction du contenu. Il est possible de lui assigner des marges (externes, internes) mais uniquement horizontales. 

a{
	display: inline; /* a est une balise inline */
}
a, span, strong, em... sont des modèles de type inline

Autres modèles

Il existe d'autre modèles de boites : none, inline-block, table, list-item, table-cell, flex...

Largeur et hauteur d'un block

Largeur : width

width permet de fixer la largeur d'un block. 
div{
	/* valeurs possibles : px, em, %... */
	width: 200px;
}

Hauteur : height

height permet de fixer la hauteur d'un block. 
div{
	/* valeurs possibles : px, em, %... */
	height: 200px;
}

Gestion des marges

Marges externes : padding

La propriété CSS margin permet de modifier les marges externes.

div{
	/* tous les cotés d'un block */
	margin: 30px;

	/* marge d'un coté : top, right, bottom, left */
	margin-top: 30px;

	/* Racourcis */
	margin: 10px 20px;  /* T&B:10px / R&L:20px */
	margin: 10px 20px 5px 0px;  /* T:10px / R:5px / BR:20px / BL:0 */
}

Marges internes : padding

padding modifie les marges internes.

div{
	/* tous les cotés d'un block */
	padding: 30px;

	/* marge d'un coté : top, right, bottom, left */
	padding-left: 30px;
}
Attention : les paddings sont ajoutées aux dimensions du block

Astuce indispensable

Pour centrer horizontalement un block, il est possible de lui donner des marges left et right à auto lorsque la largeur est fixée.

.container{
	width: 300px;
	margin: 0 auto; /* top 0, right auto, bottom 0, left auto */
}
