CSS - Background

Objectifs

Définir un background à un élément :

  • Propriété background

Propriétés - background

Couleur de fond : background-color

body{
	background-color: red; /* #F00, rgba(255, 0, 0); */
}

Image de fond : background-image

body{
	background-image: url("chemin/vers/monimage.jpg");
}

Répétition du fond : background-repeat

body{
	background-repeat: repeat; /* no-repeat, repeat-x, repeat-y... */
}

Position de l'image de fond : background-position

body{
	background-position: right top; /* top, left, right, bottom, center, %, px... */
}

Raccourci

Comme souvent en CSS (vous allez vous en rendre compte très vite), les propriétés d'un même "type" possèdent un des raccourcis pour simplifier la syntaxe.

body{
	background: blue url("../images/background.jpg") no-repeat center center;
}
