La balise PHP

Le php génère permet donc de générer des pages web. On peut insérer du PHP dans n'importe quelle structure HTML (tant que le fichier est un .php), en l'englobant des balises <?php et ?>.

<?php ... ?>
<h1>Espace membre</h1>
Bonjour <strong><?php /* pseudo */ ?></strong>

	/* un php sur plusieurs lignes */
?>
