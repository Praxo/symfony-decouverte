Balises HTML & attributs

Balises

<balise> </balise> <!-- Balise double -->
<balise /> <!-- Balise orpheline -->

Le / de fin des balises orphelines n'est plus obligatoire depuis HTML5.

Attributs : informations complémentaires

<balise attribut="valeur"> </balise>

Structure HTML de base

<!DOCTYPE html>
<html>
	<head>
		<meta charset="UTF-8">
		<title>Hello world</title>
	</head>
	<body>
		Hello world !
	</body>
</html>
  • doctype : Instruction obligatoire qui permet de définir le type de document, ses règles et sa syntaxe - dans notre cas, HTML.
  • html : Racine du document - englobe la totalité du code
  • head : Entête du document contenant toutes les méta-informations du document, son titre et les liens vers certaines ressources externes.
  • meta : Informations complémentaires (SEO - encodage - lecture du document etc...)
  • body : Le corps du document contenant l'ensembles de la partie visible du document
  • title : Le titre du document à placer dans le head

Tous ces éléments sont uniques

Attention : dans la suite du cours, je ne préciserai plus les doctype, html, head et body. Tout l'HTML sera dans la balise body.

Anciens Doctype

  • Doctype HTML 4.01 :
    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
  • Doctype XHTML 1.0
    <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">

Commentaire HTML : non visible sur le site

Les commentaires sont visibles dans le code source (CTRL + U), et dans l'inspecteur (F12), mais ils ne sont pas visibles dans la page web.
