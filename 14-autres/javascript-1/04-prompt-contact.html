Créer un prompt

Un prompt est une popin avec un champ texte à remplir. Cette valeur peut être récupérée et stockée dans une variable.

var tonAge = window.prompt("Quel age as-tu ?");
document.write(tonAge); // affiche la valeur renseignée dans le prompt

Concaténation

La concaténation consiste à ajouter une chaîne de caractères à une autre ("fusionner" des éléments).
La concaténation (en javascript), se fait avec le caractère +.

var age = 10;
document.write("J'ai : " + age + " ans"); // affichera : J'ai 10 ans

Pour concaténer des valeurs de type Integer entre elles

var age = 10;
var toto = 1;
document.write(age + "" + toto); // affichera : 101

Forcer javascript a procéder à une addition entre 2 chaines de caractères

Pour forcer une addition entre 2 chaines de caractères ayant comme valeur un nombre, nous allons utiliser la fonction parseInt() (nombre entier) ou parseFloat() (nombre entier ou décimal).

var age = "10";
var toto = "20.5";
var total = parseInt(age) + parseFloat(toto);
document.write(total); // affichera : 30.5
