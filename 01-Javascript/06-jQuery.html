Définition

jQuery est une bibliothèque JavaScript libre et multi-plateforme créée pour faciliter l'écriture de scripts côté client dans le code HTML des pages web. La première version est lancée en janvier 2006 par John Resig.

La bibliothèque contient notamment les fonctionnalités suivantes :

  • Parcours et modification du DOM (y compris le support des sélecteurs CSS 1 à 3 et un support basique de XPath) ;
  • Événements ;
  • Effets visuels et animations ;
  • Manipulations des feuilles de style en cascade (ajout/suppression des classes, d'attributs…) ;
  • Ajax ;
  • Plugins ;
  • Utilitaires (version du navigateur web…).

La bibliothèque jQuery peut être appelée de deux manières différentes :

  • via la fonction jQuery, ou $. Cette fonction est chaînable (elle retourne l'objet appelant). Par exemple :
    • $("div.test").add("p.quote").addClass("blue").slideDown("slow");
  • via l'objet $. Par exemple :
    • $.each([1,2,3], function() {
	document.write(this + 1);
});

Les sélecteurs

jQuery utilise les selecteurs CSS pour fonctionner

# pour les id, . pour les classes, etc...
Dans le cas où jQuery ne trouverait aucun élément, aucune erreur n'est renvoyé, mais un tableau vide fait office de résultat

Les fonctions

Documentation officielle

Attendre le chargement de la page 

$('document').ready(function()
{
	// Attends l'event DOMContentLoaded pour être déclenché (le chargement de la page)
});

Changer un attribut css 

$('div').css('background-color', 'red');

Changer le contenu html 

$('div').html('nouveau contenu');

Ecouter un evenement 

$('div').click(function()
{
	alert('click !');
});
