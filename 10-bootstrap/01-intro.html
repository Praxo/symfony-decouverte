Twitter Bootstrap : Framework CSS

Un framework est un ensemble de fonctions / composants pour se faciliter le travail.
Bootstrap, framework CSS responsive, va offrir un tas de classes prédéfinies.
Par exemple, on retrouvera des styles de boutons, de formulaires, de listes, de navbar... mais surtout, un système de grille fluide et responsive
Il offre également une série de plugins jQuery.

Il existe d'autres frameworks CSS comme foundation ou knacss.

NB : Je ne vais que vous introduire certaines fonctions, mais la liste est longue, soyez curieux !

Installer / Utiliser bootstrap

Télécharger le package : getboostrap.
Les fichiers minifiés sont à privilégier quand le projet est en production (gain de temps au chargement).

  • CSS : sources CSS + minifiés
    • bootstrap.css : classes de base de Bootstrap
    • bootstrap-theme.css : règles de styles pour un thème spécifique
  • fonts : polices proposées
    • glyphicons : police de pictos (Voir)
  • JS : sources JS + minifiés
