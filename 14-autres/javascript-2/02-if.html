Opérateurs conditionnels

==, !=, ===, !==, >, <, >=, <=

var number1 = 2, number2 = 2, number3 = 4, result;
result = number1 == number2;
alert(result); // true
result = number1 == number3;
alert(result); // false
result = number1 < number3; 
alert(result); // true

Les opérateurs logiques

&& (ET), || (OU), ! (NON)

Créer une condition

IF

if(true) {
	alert("message affiché");
}

if(!true) { // ou if(false)
	alert("non affiché");
}

IF / ELSE

var age = 10;
if(age > 18) {
	alert("Tu es majeur !");
} else {
	alert("Tu es mineur...");
}

IF / ELSE IF / ELSE

var age = 10;
if(age < 5) {
	alert("Tu as moins de 5 ans");
} else if( age >= 5 && age < 10) {
	alert("Tu as entre 5 et 10 ans");
} else {
	alert("Tu as 10 ans ou plus");
}

Switch

Très utile pour faire du cas par cas, et lorsqu'il y a de multiples conditions.

  • case : est suivi de la valeur qu'il doit prendre
  • break : "casse" le switch (évite d'exécuter la suite)
  • default : exécuté si aucun cas n'a été exécuté
var numeroTiroir = parseInt(prompt('Choisissez le tiroir à ouvrir (1 à 4) :'));
switch (numeroTiroir) {
    case 1:
        alert('Contient divers outils pour dessiner : du papier, des crayons, etc.');
    break;
    case 2:
        alert('Contient du matériel informatique : des câbles, des composants, etc.');
    break;
    case 3:
        alert('Ah ? Ce tiroir est fermé à clé ! Dommage !');
    break;
    case 4:
        alert('Contient des vêtements : des chemises, des pantalons, etc.');
    break;
    default:
        alert("Le meuble ne contient que 4 tiroirs et, jusqu'à preuve du contraire, les tiroirs négatifs n'existent pas.");
}

Exercice

Demander l'âge

1 à 17 ans « Vous n'êtes pas encore majeur. »
18 à 49 ans « Vous êtes majeur mais pas encore senior. »
50 à 59 ans « Vous êtes senior mais pas encore retraité. »
60 à 120 ans « Vous êtes retraité, profitez de votre temps libre ! »
Déroulement :
  • L'utilisateur est invité à renseigner son âge dans une fenêtre
  • Une fois l'âge fourni, on affiche le commentaire correspondant

Calculatrice

  1. Demander le premier nombre à l'utilisateur (peut être décimal)
  2. Demander le second nombre à l'utilisateur (peut être décimal)
  3. Demander l'opérateur ( + - * / )
  4. Afficher le résultat
