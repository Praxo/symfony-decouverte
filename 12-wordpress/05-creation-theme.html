Création d'un thème

Un thème Wordpress est un ensemble de fichiers (php, js, css, jpg...) permettant de styliser le rendu front (partie visible) d'un site Wordpress

Les thèmes Wordpress sont stockés dans /wp-content/themes/. Un thème doit être dans un dossier (actuellement il y a 3 thèmes : twentyfifteen, twentyfourteen, twentysixteen).

Les fichiers de bases

Plusieurs fichiers sont obligatoires pour la création d'un thème :

  • style.css : fichier CSS contenant des informations relatives au thème
  • index.php : fichier "template" (on reviendra dessus) par défaut

style.css

Créer un fichier style.css en indiquant ces informations en début du fichier :

/*
Theme Name: nom du thème
Theme URI: lien vers le thème
Description: description
Version: 1.0
Author: auteur
Author URI: lien vers le site de l'auteur
Tags: liste de mots clés
*/

index.php

Créer un fichier index.php en indiquant ces informations en début du fichier :

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Mon premier site Wordpress</title>
    <link rel="stylesheet" href="<?php bloginfo('stylesheet_url'); ?>" type="text/css">
  </head>
  <body>
    <h1>Bienvenue sur le site <?php bloginfo('name'); ?></h1>
    <p><?php bloginfo('description'); ?></p>
  </body>
</html>

La fonction bloginfo() permet de récupérer différentes informations relatives au site (Documentation)

  • stylesheet_url : lien vers le fichier du thème style.css
  • name : nom du site (administrable)
  • description : description du site (administrable)

Activer un thème

Avec ces 2 fichiers, le thème sera disponible dans l'administration (Apparence > Thèmes).
