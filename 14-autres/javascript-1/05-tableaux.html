Les tableaux de données - Documentation

En javascript, un tableau est une variable de type objet.
Si une variable représente une case dans la mémoire vive de l'ordinateur alors on peut définir un tableau comme une suite de cases.
Chaque case peut stocker une donnée d'un type différent.
On accède à chaque case à l'aide de son indice dans le tableau.

Indices de tableau (ou clés)

Un indice (ou une clé) correspond à la position d'une valeur dans un tableau.
Un indice peut-être numérique (par défaut) ou peut être une chaîne de caractères (on parle alors de tableau associatif).

Création d'un tableau

var tableau = new Array(); // déclaration d'une variable tableau de type objet de la classe Array
tableau[0] = "rouge"; // indice : 0 / valeur : rouge
tableau[1] = "vert"; // indice : 1 / valeur : vert
tableau[2] = "bleu"; // indice : 2 / valeur : bleu

qui revient au même que :

var tableau = ["rouge", "vert", "bleu"];
NB : Un tableau commence avec clé numérique toujours à la clé 0

Récupérer la valeur d'un tableau

var tableau = ["rouge", "vert", "bleu"];
document.write(tableau[1]); // affiche l'élément à la clé 1, donc vert
document.write(tableau[0]); // affiche l'élément à la clé 0, donc rouge

Exemple de tableau associatif

var user = new Array();
user["nom"] = "Depardieu";
user["prenom"] = "Gérard";
user["hobby"] = "Vin rouge";

document.write(user["nom"]); // affiche l'élément à la clé "nom", donc Depardieu
