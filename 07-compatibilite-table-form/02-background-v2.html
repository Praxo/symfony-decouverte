Rappels sur les background

body{
	background-color: #00aeff; /* couleur de fond (nom, hexa, rgb, rgba) */
	background-image: url("../images/backgrount.jpg"); /* image de fond (par défaut : repeat) */
	background-repeat: no-repeat; /* répétition (repeat, no-repeat, repeat-x, repeat-y) */
	background-position: right top; /* positionnement du background (top, left, right, bottom, center, px, em) */

	/* Raccourci */
	background: #00aeff url("../images/back.jpg") right top no-repeat;
}

Background size : CSS3 - Documentation

background-size est une propriété CSS3 qui permet de définir la taille du background image.
La valeur peut être (entre autre) en px, %, auto, cover ou contain. 

body{
	background-image: url("background.jpg"); /* image de fond (par défaut : repeat) */

	/* Comme c'est une propriété CSS3 */
	-webkit-background-size: cover; /* pour Chrome et Safari */
	-moz-background-size: cover; /* pour Firefox */
	-o-background-size: cover; /* pour Opera */

	background-size: cover; /* version standardisée */
}
Attention : il s'agit d'une nouvelle propriété, elle ne peut donc pas être raccourcie dans la propriété background !

Background attachment - Documentation

background-attachment permet de définir si l'image de fond est fixe ou si elle défile avec le contenu. 

body{
	background-image: url("background.jpg"); /* image de fond (par défaut : repeat) */
	background-attachment : fixed; /* valeur possible : scroll, fixed, ... */
}

  • Exemple de background-attachment fixed
