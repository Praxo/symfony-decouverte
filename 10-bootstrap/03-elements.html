Clearfix - Documentation

Bootstrap intègre par défaut la class .clearfix permettant d'englober les float (fonctionnement :after).

NB : .row intègre également la class .clearfix pour gérer les retours dans le flux

Typo iconographique - Documentation

Glyphicons est une font iconographique (partiellement gratuite) intégrée à Bootstrap, fonctionnant exactement comme font awesome, chaque icone a une classe qui lui est destinée.

<i class="glyphicon glyphicon-pencil"></i> <!-- icone avec un crayon -->

Formulaires - Documentation

  • .form-group : améliore l'espacement entre les lignes
  • .form-control : donne un style plus esthétique aux champs de formulaires
<form action="" method="post">
	<legend>Formulaire de contact</legend>
	<div class="form-group">
		<label for="test">Nom / Prénom</label>
		<input class="form-control" type="text" name="nom" id="test">
	</div>
	<div class="form-group">
		<label for="test2">E-mail</label>
		<input class="form-control" type="email" name="nom" id="test2">
	</div>
	<div class="form-group">
		<label for="test3">Message</label>
		<textarea class="form-control" name="nom" id="test3"></textarea>
	</div>
	<button>Valider</button>
</form>

Boutons - Documentation

Bootstrap fournit différents styles de boutons

<button type="button" class="btn btn-default">Default</button> <!-- Standard : blanc -->
<button type="button" class="btn btn-primary">Primary</button> <!-- Bleu -->
<button type="button" class="btn btn-success">Success</button> <!-- Vert -->
<button type="button" class="btn btn-info">Info</button> <!-- Bleu ciel -->
<button type="button" class="btn btn-warning">Warning</button> <!-- Orange -->
<button type="button" class="btn btn-danger">Danger</button> <!-- Rouge -->
<button type="button" class="btn btn-link">Link</button> <!-- Lien basique -->

Barre de navigation : navbar - Documentation

  • navbar : définit une barre de navigation
  • navbar-default : définit le style (fond noir : navbar-inverse)
<nav class="navbar navbar-default">
	<ul class="nav navbar-nav">
		<li class="active"><a href="#">Accueil</a></li>
		<li><a href="#">Présentation</a></li>
		<li><a href="#">Contact</a></li>
	</ul>
</nav>
Par défaut, la position est relative (dans le flux), mais on peut fixer la navbar avec navbar-fixed-top
