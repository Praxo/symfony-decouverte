Sélecteur descendant

Sélection des éléments contenus dans un autre élément (relation parent-enfant)

<article class="post">
	Texte noir
	<strong>Texte rouge</strong>
</article>

.post{
	color: black;
}

.post strong{
	color: red;
}

Sélecteur enfant direct

> : sélectionne les éléments directement enfants d'un élément

<article class="post">
	Texte noir
	<strong>Texte rouge</strong>
	<p>
		Contenu de mon paragraphe <strong>Texte vert</strong>
	</p>
</article>

.post strong{
	color: green;
}

.post > strong{
	color: red;
}

Sélecteur adjacent

+ : sélectionne l'élément placé directement après un élément

<article class="post">
	Texte noir
	<p>
		Contenu de mon paragraphe <strong>Texte vert</strong>
	</p>
	<strong>Texte rouge</strong>
</article>

.post strong{
	color: green;
}

.post p + strong{
	color: red;
}

États - interactions avec l'utilisateur

Il existe différents états (principalement pour les liens) :
  • :hover - état survol
  • :active - état actif (lorsqu'on clique dessus)
  • :focus - lorsque l'élément a le focus (exemple, champ texte)
  • :visited - état visité
