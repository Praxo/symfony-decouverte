Transitions CSS3

Une transition CSS3 permet une transition douce entre une valeur et une autre valeur lors d'un évènement / changement d'état (ex : hover, focus, ajout d'une class en JS...).
Plusieurs données sont nécessaires : la propriété CSS, la durée, l'interprétation et le délai

Attention : c'est du CSS3, donc pensez aux préfix (-webkit -moz ...)

Propriété(s) animée(s) : transition-property - Documentation

transition-property définit la propriété CSS qui sera animée (liste séparée par des ",").

  • all : toutes les propriétés qui ont une nouvelle valeur
  • propriété(s) : liste des propriétés (width, height, color...)
  • none : aucune
.bouton {
	transition-property: background, color; /* on animera le background et la couleur du texte */
}

Durée de la transition : transition-duration - Documentation

transition-duration définit la durée de l'animation (décimale possible).

  • s : seconde
  • ms : milliseconde
.bouton {
	transition-duration: 1.5s; /* durée de 1.5 secondes */
}

Interpolation : transition-timing-function - Documentation

transition-timing-function définit comment le navigateur interprète l'animation : accélération, décélération, linéaire...

  • linear : La vitesse est constante sur toute la durée de l'animation.
  • ease : Rapide sur le début et ralenti sur la fin.
  • ease-in : Lent sur le début et accélère de plus en plus vers la fin.
  • ease-out : Rapide sur le début et décèlere sur la fin.
  • ease-in-out : Le départ et la fin sont lents.
.bouton {
	transition-timing-function: ease-in-out;
}

Délai d'enclenchement : transition-delay - Documentation

transition-delay permet d'ajouter un délai avant la transition

  • linear : La vitesse est constante sur toute la durée de l'animation.
  • ease : Rapide sur le début et ralenti sur la fin.
  • ease-in : Lent sur le début et accélère de plus en plus vers la fin.
  • ease-out : Rapide sur le début et décèlere sur la fin.
  • ease-in-out : Le départ et la fin sont lents.
.bouton {
	transition-delay: 1s; /* retarde l'animation de 1s */
}

Raccourci

.bouton {
	background: red;
	color: #fff;
	transition: all 1.2s ease-out 1s;
}

.bouton:hover{
	background: blue;
	color: pink;
}
