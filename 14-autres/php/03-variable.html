Définir une variable

Une variable PHP doit toujours commencer par $.

<?php 
	$pseudo = "Roger"; // définit une variable pseudo avec comme valeur la chaine de caractère Roger
	echo $pseudo; // affiche la valeur de la variable $pseudo
?>

Les types de variables

Comme en JS, il y a différents types de variables

<?php 
	$pseudo = "Roger"; // string - chaine de caractères 
	$age = 18; // int - nombre entier
	$tva = 19.6; // float - décimal
	$noob = true; // booleen
	$useless = NULL; // variable "vide"
?>

Concaténation

Le caractère . permet de concaténer des variables / string

<?php 
	$pseudo = "Roger"; // string - chaine de caractères 
	$age = 18; // int - nombre entier
	echo $pseudo." a ".$age." ans !";
?>

Les opérateurs arithmétiques

+, -, *, /, %, ++, +=...

<?php 
	$age = 18; // int - nombre entier
	$age = 18 + 1;
	$age = $age + 1;
	$age += 1;
	$age ++;
?>
