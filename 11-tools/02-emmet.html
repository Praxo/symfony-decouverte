Découvrir et installer Emmet sur sublime text

Emmet est un plugin disponible pour quasiment tous les éditeurs de texte et IDE.
Il propose un tas de raccourcis pour générer de l'HTML et du CSS.

Attention : je le présente maintenant car je considère que vous maitrisez et connaissez la logique HTML et les balises (imbrication / attributs), ainsi que la syntaxe des propriétés CSS.
Si ce n'est pas encore totalement le cas, privilégiez l'écriture en dur !

Installer Emmet sur sublime text

  • CTRL + Shift + P : ouverture de la palette de commandes
  • Recherchez Install package puis "enter" (Si vous ne le trouvez pas, appelez moi - Installation de la palette de commandes)
  • Recherchez Emmet puis "enter" pour lancer l'installation

Raccourcis HTML

Il faut être dans un fichier HTML pour les utiliser.
Tous les raccourcis vont être déclenché en utilisant "tab".

.toto : balise générique avec class

<div class="toto"></div>

nav.toto : balise spécifique avec class

<nav class="toto"></nav>

ul>li : relation parent - enfant

<ul>
	<li></li>
</ul>

header+main : relation frère

<header></header>
<main></main>

nav>ul>li*2>a : relation parent - enfant multiple

<nav>
	<ul>
		<li><a href=""></a></li>
		<li><a href=""></a></li>
	</ul>
</nav>

Raccourcis CSS

Il faut être dans un fichier CSS pour les utiliser.
Tous les raccourcis vont être déclenché en utilisant "tab".

w20

width: 20px;

tac

text-align: center;

fwb

font-weight: bold;

dib

display: inline-block;
