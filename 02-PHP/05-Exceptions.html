HTML - listes

Objectifs

Créer des listes

  • Liste non ordonnée (puces)
  • Liste ordonnée (numérique)

Liste non ordonnée

<ul>
	<li>HTML</li>
	<li>CSS</li>
	<li>Javascript</li>
</ul>
  • <ul> </ul> : délimite toute une liste non ordonnée
  • <li> </li> : délimite un élément de la liste

Exemple

  • HTML
  • CSS
  • Javascript

Liste ordonnée

<ol>
	<li>Obi wan kenobi</li>
	<li>Chewbacca</li>
	<li>Dark Vador</li>
	<li>La réponse D</li>
</ol>
  • <ol> </ol> : délimite toute une liste ordonnée
  • <li> </li> : délimite un élément de la liste

Exemple

  1. Obi wan kenobi
  2. Chewbacca
  3. Dark Vador
  4. La réponse D

Différents niveaux de liste

Il est possible de créer des sous-listes 

<ul>
	<li>
		Intégration
		<ul>
			<li>HTML</li>
			<li>CSS</li>
		</ul>
	</li>
	<li>
		Développement
		<ul>
			<li>Javascript</li>
			<li>PHP</li>
			<li>SQL</li>
		</ul>
	</li>
</ul>

Exemple

  • Intégration
    • HTML
    • CSS
  • Développement
    • Javascript
    • PHP
    • SQL
