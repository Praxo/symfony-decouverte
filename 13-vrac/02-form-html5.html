<input type="color" name="colorPicker" />

Compatibilité entre les navigateurs

HTLM5shim permet aux anciennes versions de navigateurs de reconnaître et interpreter les éléments HTML5.
Il suffit de placer ce bout de code dans le head de votre document HTML. 
<!--[if lt IE 9]>
	<script src="http://html5shim.googlecode.com/svn/trunk/html5.js"></script>
<![endif]-->
