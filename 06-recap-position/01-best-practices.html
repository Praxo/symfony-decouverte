Best practices - HTML / HTML5

  • Réfléchir et faire un schéma sur papier avant de commencer à coder. Toujours commencer par ce que l'on sait faire
  • Commencer par la structure HTML avant de commencer le CSS
  • Bien indenter, structurer, commenter son code (comment ça, je ne le répète pas assez ?)
  • Commencer par le positionnement des blocs, le faire rangée de blocs par rangée de blocs
  • Toujours préciser l'attribut alt des images (vide si l'image n'a pas de sens particulier)
  • Préférez les class aux id
  • Préférez les valeurs fluides aux fixes (% / em / rem)
